This Dashboard uses HTML/CSS/PHP/Javascript Bootstrap & MySQL

**Script is for demonstration/learning purposes only.  Please limit this to your internal network!

Installation:
=============

1. Copy contents of folder “dashboard” to your web directory, must have PHP 5.4 or greater!

2. Create a database using phpMyAdmin on your host.  Use the “Import” utility to import srvr_status.sql.  It is a single table.  There is some mock data in the database.  While in phpMyAdmin, go ahead and update the fields to Servers you want to monitor.

3. If your execution scripts are running, you should see the timestamp on the dashboard update!
